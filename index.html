Using Ethplay, anyone can create and host games on Ethereum

Operating a game on Ethereum is as easy as sending a transaction.
A system of smart contracts allows users to design new games, host existing games, and profit from referals and advertisement.
Game designers can focus on designing a great game, instead of having to tackle complicated blockchain programming tasks.


The Ethplay Token (PLY) offers dividend functionality, in-game utility, and platform governance.

Token holders are the true owners of Ethplay. The Ethplay token is the first to offer dividend functionality. All profits are distributed to token holders as dividends. Tokens can be used as in-game currency. Platform governance functions, such as setting network parameters, are accessible to token holders. The total amount of tokens is capped at 100 million.
